This directory contains the following programmatically generated files:

- `arc_t3.bin`, the round constants for the ARC layer of Poseidon(T=3)
- `arc_t4.bin`, the round constants for the ARC layer of Poseidon(T=4)
- `g1.bin`, a KZG SRS containing 65536 $G_1$ points
- `g2.bin`, a KZG SRS containing 65536 $G_2$ points

`g1.bin` and `g2.bin` have been generated by the [`generate_params`][generate_params] script.

`arc_t3.bin` and `arc_t4.bin` have been generated by manually adapting the
[official Rust code][poseidon2-params] for the BLS12-381 instance.

[generate_params]: https://github.com/libernet-mirror/generate_params
[poseidon2-params]: https://github.com/HorizenLabs/poseidon2/blob/main/plain_implementations/src/poseidon2/poseidon2_params.rs
